## 🛠️ Tech Stack

- **HTML5** — Semantic, single-file structure
- **CSS3** — Custom properties (CSS variables), Grid, Flexbox, animations, responsive media queries
- **Vanilla JavaScript** — Scroll-based nav highlighting, IntersectionObserver for reveal animations, mobile hamburger menu
- **Google Fonts** — [Syne](https://fonts.google.com/specimen/Syne) (headings) + [DM Sans](https://fonts.google.com/specimen/DM+Sans) (body)

No build tools, no dependencies, no frameworks — just a single `.html` file.

---

## ✨ Features

- **Animated hero** with floating gradient orbs and staggered fade-up entrance
- **Scroll-reveal animations** powered by `IntersectionObserver`
- **Animated skill bars** that fill on scroll-into-view
- **Sticky navbar** with blur/glassmorphism effect on scroll
- **Active section highlighting** in the nav as you scroll
- **Mobile hamburger menu** with fullscreen overlay
- **Noise texture overlay** for a premium feel
- **Smooth scrolling** throughout the page
- **Fully responsive** — works on desktop, tablet, and mobile

## 🚀 Getting Started

Since this is a single HTML file, there's no setup required.

```bash
# Clone the repository
git clone https://github.com/bhavinprajapat/portfolio.git

# Open in browser
open bhavin-portfolio.html
```

Or simply download and double-click the `bhavin-portfolio.html` file.

---

## 🌍 Deployment

This site can be deployed anywhere that serves static files:

- **GitHub Pages** — Push to `main` and enable Pages in repo settings
- **Netlify** — Drop the HTML file at [netlify.com/drop](https://netlify.com/drop)
- **Vercel** — `vercel deploy`
